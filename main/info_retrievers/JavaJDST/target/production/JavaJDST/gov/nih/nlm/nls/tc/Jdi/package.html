This package provides base Java classes of all Jdi operations. It includes classes to:

  • read file and translate into java objects for
    • contractions
    • Journal descriptors
    • legal words
    • sub headings
    • Word Signal Wc Dc
    • JidTaJd
    • JidTaJds
  • Jd score comparators
    • Count1fJdComparator
    • Count2fJdComparator
    • WordSignalWcDcComparator

This package is used in the text categorization tools and is not intended to be used by end users directly. End users may skip this package.
