This package provides Java APIs classes. There are five major application programming inrterface in this package. They are briefly described as follows:

  • Java APIs Class:
    • JdiApi: Journal Descriptor Indexing API
    • MltApi: MEDLINE tokenizer API
    • SimilarityApi: Calculate similarity for JD Vectors or ST Vectors
    • StiApi: Semantic Type Indexing API
    • StriApi: Semantic Type Realtime Indexing API
    • StWsdApi: ST WSD API
