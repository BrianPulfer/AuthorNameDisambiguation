This package provides base Java classes of all Stri operations. It includes classes to:

  • read file and translate into java object for
    • StJd
    • StJds
  • StJd scores comparator and StJd comparator
    • StJdComparator
    • Count1fStComparator

This package is used in the text categorization tools and is not intended to be used by end users directly. End users may skip this package.
